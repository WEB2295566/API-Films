API Films - Documentation

GitHub

Cette API permet d'afficher, d'ajouter, de modifier et de supprimer des films de la base de données. Les routes sont accessibles à partir de l'URL suivante :
https://api-films-bbmz.onrender.com/api/films/

Routes

Récupérer la liste de tous les films

GET api/films/

https://api-films-bbmz.onrender.com/api/films/

Paramètres possibles

https://api-films-bbmz.onrender.com/api/films?tri=realisation&ordre=asc

Tri
  • titre
  • annee
  • realisation
Ordre
  • asc
  • desc

Récupérer un film spécifique

GET api/films/{id}

https://api-films-bbmz.onrender.com/api/films/p9ruH1HhIGni7pNBB6Gv

Paramètre requis : id (l'identifiant du film à récupérer).

Ajouter un nouveau film

POST api/films

https://api-films-bbmz.onrender.com/api/films/

{
    "titre": "Titre du film",
    "description": "Description du film",
    "annee": 2024,
    "realisation": "Nom du réalisateur",
    "genre": ["Genre 1", "Genre 2"],
    "titreVignette": "Titre de la vignette"
}

Modifier un film existant

PUT api/films/{id}

https://api-films-bbmz.onrender.com/api/films/{id} 

{
    "titre": "Titre du film",
    "description": "Description du film",
    "annee": 2024,
    "realisation": "Nom du réalisateur",
    "genre": ["Genre 1", "Genre 2"],
    "titreVignette": "Titre de la vignette"
}

Supprimer un film

DELETE /api/films/{id}

https://api-films-bbmz.onrender.com/api/films/{id}

Paramètre requis : id (l'identifiant du film à supprimer).

Routes pour les utilisateurs

Inscription d'un nouvel utilisateur

POST /api/utilisateurs/inscription

https://api-films-bbmz.onrender.com/api/utilisateurs/inscription 

{
    "courriel": "test@test.com",
    "mdp": "Ab1234!@hello"
}

Les critères de complexité pour le mot de passe sont détaillés ci-dessus.

La validation du mot de passe exige qu'il soit long de 8 à 20 caractères et qu'il combine lettres (minuscules et majuscules), chiffres, et symboles spéciaux. Cette règle garantit un niveau de sécurité élevé en rendant le mot de passe complexe à deviner.

Connexion d'un utilisateur

POST api/utilisateurs/connexion

https://api-films-bbmz.onrender.com/api/utilisateurs/connexion 

{
    "courriel": "test@test.com",
    "mdp": "Ab1234!@hello"
